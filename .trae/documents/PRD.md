# 产品需求文档（PRD）— 奢侈品包包展示网站

## 1. 产品概述
本项目是一个面向海外客户的高端奢侈品包包（1:1 复刻）展示型网站，定位为"轻量级商品目录 + 询价转化"。
- 解决问题：将抓取的商品素材（约 6000+ SKU、17 个品牌）以精致、可信的视觉呈现，引导客户通过 WhatsApp 询价下单，无需在线交易。
- 目标用户：寻找高端复刻奢侈包的海外买家；核心价值是用高级感设计建立信任、提升询价转化率。
- 部署目标：Vercel 静态托管，零服务器成本，全球 CDN 加速。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问方式 | 权限 |
|------|----------|------|
| 访客 | 直接访问 | 浏览全部商品、品牌、详情、搜索、WhatsApp 询价 |
| （无后台/无注册，纯展示型站点） | — | — |

### 2.2 功能模块（页面）
1. **首页**：品牌 Hero、精选商品、品牌矩阵、工艺/卖点区块、询价引导
2. **全部商品页**：商品网格、品牌/品类筛选、排序、分页
3. **商品详情页**：多图画廊、规格、描述、相关推荐、WhatsApp 询价
4. **品牌页**：品牌头图、品牌全部商品（筛选+分页）
5. **搜索**：全局搜索弹窗（按名称/品牌实时检索）
6. **内容页**：关于我们、联系我们、物流政策、退换政策、尺码指南
7. **询价转化**：WhatsApp 浮动按钮 + 商品页询价按钮 + 联系表单

### 2.3 页面明细
| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 | Hero 区 | 全屏品牌大图 + 标语 + CTA，渐入动画 |
| 首页 | 精选商品 | LV/Chanel/Dior 等精选商品网格，hover 切换图 |
| 首页 | 品牌矩阵 | 17 个品牌卡片，显示商品数量，点击进入品牌页 |
| 首页 | 卖点区 | 全球免邮、30天退换、1:1 品质、安全包装 |

| 搜索 | 搜索弹窗 | 实时检索，结果直跳详情页 |

## 3. 核心流程

**主浏览流程**：首页 → 浏览精选 → 进入品牌/全部商品 → 筛选 → 商品详情 → WhatsApp 询价

**搜索流程**：点击搜索 → 输入关键词 → 结果列表 → 点击进入详情 → 询价

## 4. 用户界面设计

### 4.1 设计风格
- **风格定位**：奢华编辑风（Editorial Luxury）— 克制、留白、高级感，避免廉价感
- **主色**：暖象牙白底（#FAF8F5）+ 墨黑（#1A1A1A）文字 + 香槟金（#B8956A）点缀
- **强调色**：香槟金用于按钮/线条/hover；WhatsApp 询价用品牌绿（#25D366）
- **字体**：标题用衬线字体（Cormorant Garamond / Playfair Display）显奢华；正文用简洁无衬线（Manrope）
- **按钮风格**：细线框 + 实心黑两种；圆角小（4px）；hover 微位移
- **布局**：宽屏留白、大图主导、网格规整、顶部固定导航
- **图标**：Lucide 线性图标，纤细风格
- **动效**：页面载入渐入、图片 hover 缩放、卡片 hover 上浮，克制不浮夸

### 4.2 页面设计概览
| 页面 | 模块 | UI 元素 |
|------|------|---------|
| 首页 | Hero | 全屏大图、衬线大标题、香槟金细线、CTA 黑色按钮 |
| 首页 | 商品卡 | 白底卡片、商品图、品牌小字、hover 显示第二张图 |
| 全部商品 | 筛选栏 | 左侧抽屉式筛选、顶部排序下拉、商品网格 |
| 详情页 | 画廊 | 左侧大图 + 竖排缩略图、右侧信息栏 |
| 详情页 | 询价 | 香槟金 WhatsApp 大按钮 + 浮动按钮 |

### 4.3 响应式
- 桌面优先（1280px+），平板自适应（768px+），移动端优化（<768px 单列、汉堡菜单、底部询价栏）
- 触屏优化：图片可滑动、按钮加大点击区域

## 5. 数据说明
- 数据来源：`yutulu_scraper_data/*.json`（构建时转换为静态 JSON）
- 商品字段：id、slug、name、description、categories、images（r2Url 云存储直链）
- 图片：直接引用 R2 云存储 URL，无需本地存储
- 品牌（17）：Louis Vuitton、Chanel、Dior、Gucci、Hermes、YSL、Prada、Bottega Veneta、Celine、Goyard、Loewe、MiuMiu、Balenciaga、Fendi、Loro Piana、Givenchy、Burberry
